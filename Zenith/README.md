# E-SUMMIT 2026 Website

A retro-desi themed website for E-SUMMIT 2026 (MAIT | Communite Ã— EDC).

## Tech Stack
- React 19
- Tailwind CSS 4
- Framer Motion

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to the local URL (usually http://localhost:5173).

## Features
- **Retro Desi Theme**: Custom colors, fonts (Syne, Bebas Neue), and grain textures.
- **Hero Section**: Parallax background, animated countdown.
- **Events**: Horizontal scroll interaction.
- **Responsive Design**: Mobile-first approach.

## Structure
- `src/components/`: Reusable components (Hero, Navbar, About, Events, Contact).
- `src/index.css`: Global styles and Tailwind configuration.
